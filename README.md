# 📊 Web-Based Personal Finance Tracker

A sleek, minimalist, and fully interactive personal finance tracker built as a single-page web application. This tool allows for quick logging of income, expenses, and cash flow right from your browser.

## ✨ Features
* **Interactive Dashboard:** Real-time visual updates of your financial standing.
* **Aesthetic Categories:** Clean, color-coded tracking for easy scanning (Home, Savings, Spiritual, etc.).
* **Zero Installation:** Runs entirely in any modern web browser without needing a backend server.

## 🛠️ Built With
* **HTML5:** Semantic structure for the tracker interface.
* **CSS3:** Clean, minimalist, and responsive layout design.
* **JavaScript (ES6):** Dynamic calculations and data manipulation.

## 🚀 How to Run Locally
1. Clone or download this repository.
2. Locate the `index.html` file in the project folder.
3. Double-click `index.html` to open and use the tracker instantly in your browser.
